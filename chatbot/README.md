# 🔭 NOVA — Nebula-Oriented Virtual Astronomer

An **Agentic RAG Chatbot** specialized in Space Exploration & Astronomy.

Built with Flask · OpenAI · ChromaDB · LangChain · Pure HTML/CSS/JS

## Quick Start

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env   # Add your OPENAI_API_KEY
python main.py
# Open http://localhost:5000
```

See [`instructions.md`](instructions.md) for full setup and architecture details.
